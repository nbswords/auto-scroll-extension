# Auto Scroll

A Chrome extension that automatically scrolls web pages with a natural, human-like rhythm — scroll a distance, pause, then repeat.

[Chrome Extension Link](https://chromewebstore.google.com/detail/auto-scroll/cclcciliebgalipkdjgiiekkachkemio)

## Features

- **Natural Scrolling** — Scrolls in intervals with configurable pauses, mimicking human mouse wheel behavior
- **Adjustable Speed** — Control how fast each scroll segment moves (1–10)
- **Custom Distance** — Set how far each scroll moves (100–1000px)
- **Pause Duration** — Configure how long to pause between scrolls (0.5–10 sec)
- **Direction Control** — Scroll down or up
- **Light & Dark Theme** — Toggle between themes, preference is saved
- **Privacy First** — Zero data collection, everything runs locally

### Manual Install
1. Download or clone this repository
2. Open `chrome://extensions/` in Chrome
3. Enable **Developer mode** (top right)
4. Click **Load unpacked**
5. Select the project folder

## Usage

1. Click the Auto Scroll icon in the toolbar
2. Adjust speed, distance, and pause duration
3. Click **Start Scrolling**
4. Click **Stop Scrolling** to pause at any time

## Permissions

| Permission | Reason |
|------------|--------|
| `activeTab` | Access the current tab only when you click the extension icon |
| `scripting` | Inject the scroll script on-demand when you start scrolling |

No background permissions. No data leaves your browser.
